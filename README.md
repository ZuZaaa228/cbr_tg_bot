## Установка:

1. Клонируйте репозиторий:
   ```bash
   git clone https://github.com/ZuZaaa228/cbr_tg_bot.git
   ```
2. Создайте файл .env:
   Скопируйте содержимое примера `.env` и замените `TELEGRAM_API_TOKEN` на ваш токен API Telegram.
3. Запустите приложение:
   ```bash
   docker-compose up -d
   ```
